# Proyecto: Manipulación de Datos y Operaciones con CSV usando Python

Este proyecto consta de tres scripts en Python que manejan la manipulación de datos y la combinación de archivos CSV y Excel utilizando la biblioteca `pandas`. A continuación, se presenta un resumen de la funcionalidad de cada script.

## Requisitos Previos

* Python 3.x
* Biblioteca Pandas (`pip install pandas`)

---

## 1. `combinarDosCsv.py`

### Descripción

Este script combina datos de un archivo Excel y un archivo CSV realizando una operación de fusión (merge) basada en una clave común (`id_usuario`).

### Funcionalidad

* Lee un archivo Excel llamado `Usuarios.xlsx` desde el directorio `Sheet Originales`.
* Lee un archivo CSV llamado `UsuariosEditados.csv` desde el directorio `Sheet Editados`.
* Combina los dos dataframes usando la columna `id_usuario` como clave.
* Genera un archivo CSV combinado llamado `Merge Viajes - Usuarios.csv`.

## 2. `lecturaViajes-Usuarios.py`

### Descripción

Este script lee un archivo CSV combinado que contiene datos de viajes y usuarios, procesa varias columnas, y genera un nuevo archivo CSV con las modificaciones aplicadas.

### Funcionalidad

* Lee un archivo CSV llamado `Merge Viajes - Usuarios.csv` desde el directorio `Sheet Combinados`.
* Convierte ciertos campos a tipo `str` para realizar modificaciones.
* Realiza varias operaciones de limpieza y transformación, como:
  * Eliminar sufijos específicos de IDs.
  * Modificar valores de columnas (ej. traducir géneros, ajustar coordenadas).
  * Eliminar columnas innecesarias (`fecha_alta`, `hora_alta`, `modelo_bicicleta`).
* Exporta el archivo CSV modificado como `Viajes y Usuarios Editado.csv`.

## 3. `leecturaEstaciones.py`

### Descripción

Este script lee un archivo CSV que contiene datos de estaciones, realiza algunas verificaciones y modificaciones, y puede exportar los datos modificados.

### Funcionalidad

* Lee un archivo CSV llamado `Estaciones.csv` desde el directorio `Sheet Originales`.
* Permite la visualización de las columnas leídas.
* Filtra datos según criterios específicos (ej. elimina filas con valores mayores a 1 en la columna `LINEA DE PEDIDO`).
* Opcionalmente, exporta los datos filtrados a un archivo `.xlsx` (comentado por defecto en el código).
